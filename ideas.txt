  [x] 0x00 Null
        0x10 End
  [x] 0x01 Byte            i8
  [x] 0x02 Short           i16
  [x] 0x03 Int             i32
  [x] 0x04 Long            i64
  [x] 0x05 Float           f32
  [x] 0x06 Double          f64
  [x] 0x07 String          utf8
  [x] 0x08 Map
  [x] 0x09 List
  [x] 0x0A ByteArray       i8[]
  [x] 0x0B ShortArray      i16[]
  [x] 0x0C IntArray        i32[]
  [x] 0x0D LongArray       i64[]
  [x] 0x0E FloatArray      f32[]
  [x] 0x0F DoubleArray     f64[]

Additional Data

  List/ByteArray/ShortArray/IntArray/LongArray/FloatArray/DoubleArray
    0x00 Read list length as uint8
    0x10 Read list length as uint16
    0x20 Read list length as int32
    0x30 Indefinite length

  Null
    0x10 Interprets it as an End "code". Used for indefinite length list/array/map
